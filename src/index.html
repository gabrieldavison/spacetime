--esc-- - toggle configuration screen

--space-- - stop/start sequencer

1/2/3/4 - switch sequencer preset

left/right arrows - move edit position

up/down arrows - cycle through commands

Commands:

  • + increments current note by one
  • - decrements current note by one
  • < sets current note to the lowest note in the scale
  • > sets current note to the highest note in the scale
  • * sets current note to a random note in the current octave
  • f sets the metronome to fast speed
  • s sets metronome to slow speed
  • # skips to a random position in the sequence
  • _ skips current step
  • = plays current note (Same as previous step)
